## 3. Files Used

The project consists of three main files:

### 1. `index.html`
* Contains the structure of the webpage.
* Uses semantic HTML5 elements.
* Organizes quiz content into sections.

### 2. `style.css`
* Handles styling and layout.
* Includes animations and transitions.
* Ensures responsive design using media queries.
* Implements modern UI effects.

### 3. `script.js`
* Stores quiz questions in a JavaScript array.
* Handles option selection.
* Checks answers and updates score.
* Manages timer functionality.
* Controls progress bar updates.
* Displays final results.

This separation of files improves maintainability and code organization.

## 4. Choices Made

Several important design and development choices were made:

### 1. JavaScript Array for Questions
Questions are stored in a JavaScript array to allow easy editing and admin control.

### 2. Responsive Design
Used flexible widths and media queries to ensure compatibility with mobile devices.

### 3. Timer Feature
Added a 15-second timer per question to increase engagement.

### 4. Randomization
Questions are shuffled each time the quiz loads to make the experience dynamic.

### 5. CSS Animations
Used transitions and keyframe animations to:
* Smoothly display feedback
* Animate background
* Enhance user interaction

### 6. Accessibility Considerations
* Used semantic HTML elements
* Added ARIA roles
* Maintained readable contrast and font sizes

These choices improve usability, interactivity, and overall performance.
